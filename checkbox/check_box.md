## Checkbox 
- [x] ticked
- [ ] unticked

## Nested second-level check box

### Good to know

Pressing the tab key will nest the checkbox under the parent element.

Do not use `- -`
If you use `- -`, it will create a bullet point with a checkbox instead of a nested checkbox.
  
### Wrong
- [x] ticked
- - [x] ticked
- - [ ] unticked
- - - [x] ticked
- - - [ ] unticked


### Correct
- [x] ticked
  - [x] ticked
  - [ ] unticked
    - [x] ticked
    - [ ] unticked
